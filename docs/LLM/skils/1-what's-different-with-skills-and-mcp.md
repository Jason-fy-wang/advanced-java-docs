#### what's the different with skills and mcp?
Simple, MCP is kind of tool, which extend the functional of model.  Usually, model with leverage the MCP to get the specific message (information).

Skills:  skills tells about model how to process the message.  And skills gives the `message format`, `reference docs`,`script`.  And `msg format` tells model how to generate message, `reference` tells model what documents it can refer, `script` which model can execute the script to process the message.
