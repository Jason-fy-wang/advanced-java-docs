# gradle常用的命令

## 1. 查看所有的task

```shell
# 查看所有的tasks
gradle  tasks
gradle	tasks --all
```



## 2. 查看某一个task的帮助

```shell
# 查看某个task的帮助,只需要把 换上对应task的name
gradle	help   --task  taskName
```
