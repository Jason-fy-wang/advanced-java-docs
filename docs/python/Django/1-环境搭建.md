# Django 环境搭建

## 1. 创建virtualenv

```shell
virtualenv -p python3 Django_3.2
```





## 2. 安装对应的软件包

```shell
pip install Django==3.2 -i https://pypi.douban.com/simple
```





## 3. 创建项目

```shell
django-admin.exe  startproject  WebServer
```
