# 变量的使用

## 1. 变量的定义
### 1.1 变量的三种定义
```go
// 第一种方式, 定义但不初始化
var a int
``
// 第二种方式, 定义及初始化
var b int = 2
// 此处省略的类型, go可以自行推断
var b = 3

// 第三种, 定义并初始化
// 此处省略了 var和类型, 类型go会自行推断
// 符号 :=  表示同时声明和初始化了变量
c := 4
```

### 1.2 一次定义多个变量
```go
// 定义多个变量, 但不进行初始化
var a,b,c int

// 定义多个变量，并进行初始化
var a, b, c = 1, 2, 3
// 定义多个变量并初始化
a , name, c = 1, "tom", 3

```
