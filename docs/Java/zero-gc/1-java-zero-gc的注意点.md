#### java zero-gc代码的常用思路
> 1. 使用堆外内存, 自己进行内存管理
> 2. 使用对象池, 尽量让对象复用
> 3. 进行`逃逸分析`, 让对象 分配到栈上.
> 4. 针对 map  list set等 容器进行复写, 使用 int long 等primitive type, 不使用对象
> 5. 针对需要持久化的对象, 使用 mmap
